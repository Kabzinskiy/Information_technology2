# Information_technology2
По результатам, полученным в предыдущей лабораторной работе, построить коды сжатия информации используя метод Шеннона-Фэно (для тех у кого последняя цифра но-мера зачетки – четная) или метод Хаффмена (для тех у кого эта цифра – нечетная). По-строение произвести, создав программу на языке Pascal или С.
Имя результирующего файла аналогично имени файла, в котором хранились часто-ты, но расширение .S_F если кодировка проводилась методом Хаффменна, или .S_F если кодировка прово-дилась методом Шеннона-Фэно.
Например, обрабатывая файл doc01.tab методом Хаффмена получим файл с кодо-вой табли-цей с именем doc01.S_F, если же этот файл обработан методом Шеннона-Фэно, то в результате полу-чим файл с именем doc01.S_F.
Содержание результирующего файла:
1. первая строка – имя исследуемого информационного файла;
2. вторая строка – количество байт в исследуемом файле;
3. пустая строка;
4. последующие строки – коды символов и коды сжатия, разделенные
пробелом или символом табуляции;
5. пустая строка
6. средняя длинна кода сжатия
7. последняя строка – величина энтропии.
